HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title changes title of webpage
  • style element css styles associated with website
  • base is the URL of page
  • link connects an external source to HTML doc
  • meta character set, description, characteristics, and viewport
  • script embeds client side scripts in the html doc
  • head is a container for meta
  • br creates lines of empty space

CSS

  • a margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • a programming language that adds interactivity to your website.
  • APIs Application Programming Interfaces: provide functionality
  • Variables: containers that store values
  • String: This is a sequence of text known as a string.
  • Array: This is a structure that allows you to store multiple values in a single reference.
  • Conditionals are code structures used to test if an expression returns true or not.
  • The null value represents the intentional absence of any object value.
